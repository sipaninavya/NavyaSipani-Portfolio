Featured Project In Development

Cipher
Stones

// Top-down mobile grid puzzle · Unity 6.3 LTS

A swipe-based mobile puzzle game built around three nested gameplay loops — micro (solving a grid puzzle), mid (completing a zone), and meta (uncovering a hidden five-letter word across all five zones). The loop structure is designed from the ground up to motivate through intrinsic satisfaction, deliberately cutting features like daily streaks and push notifications that trade long-term engagement for short-term retention.

Core systems include a par-move and undo system (full game-state history via a Stack), a 3-star rating per level, ScriptableObject-based level data, and a Tilemap/Grid architecture. Zone layouts are being paper-playtested before any implementation begins — validating par-move counts and difficulty progression before a single line of Unity is written.

Engine Unity 6.3 LTS
Platform Mobile (Android)
Zones 5 (20 levels)
Architecture Tilemap / Grid · ScriptableObjects
Undo System Stack-based history
Status Design & paper prototyping
